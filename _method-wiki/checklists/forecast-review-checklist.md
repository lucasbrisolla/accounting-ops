# Checklist de Revisão de Forecast

## Objetivo

Revisar se o forecast é uma visão atualizada e defensável do resultado provável.

Para uma revisão mais ampla de qualquer projeção financeira, use também `financial-projection-quality-checklist.md`.

## Checklist

- [ ] O horizonte do forecast está claro.
- [ ] O comparativo principal está definido: budget, forecast anterior ou operating plan.
- [ ] Os principais drivers foram identificados.
- [ ] As premissas críticas estão explícitas.
- [ ] O forecast parte do realizado e da tendência recente.
- [ ] Cenário base, upside e downside estão separados.
- [ ] Riscos e oportunidades têm gatilhos de monitoramento.
- [ ] Ações gerenciais estão associadas aos principais desvios.
- [ ] A projeção não está apenas repetindo o orçamento original.
- [ ] Incertezas relevantes foram declaradas.
- [ ] A projeção foi comparada com tendência histórica, run rate ou extrapolação simples.
- [ ] Sensibilidades relevantes foram testadas.
- [ ] Downside com alta probabilidade foi avaliado para entrar no cenário base.

## Perguntas de revisão

1. O forecast representa melhor estimativa ou meta desejada?
2. O que mudou desde o último forecast?
3. Quais premissas foram invalidadas?
4. O downside é maior que o upside?
5. O time operacional assumiu ownership das premissas?
6. Que driver justifica qualquer reversão forte contra a tendência recente?
