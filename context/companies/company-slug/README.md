# Company Slug

Pasta pronta para iniciar a primeira destilação real.

## Antes de usar

1. Renomear `company-slug/` para o `slug` real da companhia.
2. Atualizar os títulos e placeholders dos arquivos.
3. Preencher `source-manifest.md`.
4. Posicionar a primeira fonte em `raw/`.

## Escopo desta pasta

Guardar apenas:

- fontes da companhia
- leituras e evidências rastreáveis
- contexto operacional para a empresa piloto

Não usar esta pasta para método geral.
