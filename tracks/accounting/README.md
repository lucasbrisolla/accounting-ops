# Trilha Accounting

Trilha reservada para accounting e controladoria dentro do `accounting-ops`.

## Papel

Concentrar conteúdos de:

- fechamento
- qualidade do número
- conciliações
- controladoria gerencial
- políticas e rotinas contábeis relevantes para gestão

## Estrutura atual

- `modes/accounting-closing-and-quality.md`
- `modes/accounting-technical-application.md`
- `workflows/monthly-closing-and-number-quality.md`
- `workflows/account-reconciliation-review.md`
- `workflows/working-capital-and-cash-conversion-review.md`
- `workflows/cash-management-and-bank-reconciliation.md`
- `workflows/inventory-and-valuation-review.md`
- `playbooks/result-reading-and-number-quality.md`
- `playbooks/cpc-ifrs-impact-on-management-number.md`

## Princípio

O foco inicial desta trilha é híbrido:

- mais peso em fechamento, controladoria gerencial e qualidade do número
- com camada técnica suficiente para explicar quando CPC/IFRS muda a leitura do resultado

O objetivo é permitir crescimento modular sem misturar tudo na camada central.
